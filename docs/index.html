Flexbox Enabled

Many of Figuration's components, but not all, are built using flexbox. This means enhanced layout and alignment control, but it does come at the cost of not being able to support older browsers.

Sass Source

Figuration compiles and uses standard CSS for it's output, but on the source side we use the power of Sass and Autoprefixer for faster and consistent development.
